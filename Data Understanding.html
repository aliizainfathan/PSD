Data Understanding#

Sumber Data#

Dataset diambil dari link dibawh ini:

https://huggingface.co/datasets/farhamu/tokopedia-product-reviews-2019

Dataset tokopedia-product0reviews-2019 merupakan kumpulan data review pelanggan dari produk pada masing-masing kategori. Kumpulan data ini berisi 40.607 ulasan produk dari Tokopedia, salah satu platform e-commerce terbesar di Indonesia, yang dikumpulkan pada tahun 2019. Kumpulan data ini memberikan wawasan berharga tentang sentimen pelanggan dan perilaku belanja di pasar e-commerce Indonesia.

Pengambilan data#

Mengambil data pada site Hugging Face, telah disediakan berbagai cara unutuk mengakses dataset di antaranya melalui library python seperti Dataset, Pandas, Croiisant, dan Pollars. Berikut cara mengambil data dan menampilkan data yang diambil dari sites Hugging Face:

from datasets import load_dataset
import pandas as pd

# Ambil dataset dari Hugging Face
dataset = load_dataset("farhamu/tokopedia-product-reviews-2019")

df = dataset["train"].to_pandas()

print(display(df.head()))

Fitur dalam Dataset#

df.columns

Index(['text', 'rating', 'category', 'product_name', 'product_id', 'sold',
       'shop_id', 'product_url'],
      dtype='object')

Berdasarkan data di atas fitur yang terdapat pada dataset tersebut yakni:

  1. text – Ulasan pelanggan dalam bentuk teks.

  2. rating – Penilaian numerik pelanggan (biasanya 1–5).

  3. category – Kategori produk yang diulas (misal: Elektronik, Fashion).

  4. product_name – Nama produk yang diulas.

  5. product_id – ID unik produk di Tokopedia.

  6. sold – Jumlah produk yang telah terjual.

  7. shop_id – ID unik toko penjual produk.

  8. product_url – URL halaman produk di Tokopedia.

Berdasarkan hasil tabel di atas bahwa setiap fitur memiliki tipe data di antaranya:

  1. text – Berupa tipe data objek yakni dapat berupa string atau gabungan antara string dan integer.

  2. rating – Memiliki tipe data Integer dengan panjang 64.

  3. category – Berupa tipe data objek yakni dapat berupa string atau gabungan antara string dan integer.

  4. product_name – Berupa tipe data objek yakni dapat berupa string atau gabungan antara string dan integer.

  5. product_id – Memiliki tipe data Integer dengan panjang 64.

  6. sold – Berupa tipe data objek yakni dapat berupa string atau gabungan antara string dan integer.

  7. shop_id – Memiliki tipe data Integer dengan panjang 64.

  8. product_url – Berupa tipe data objek yakni dapat berupa string atau gabungan antara string dan integer.
